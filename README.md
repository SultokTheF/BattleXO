# Real-Time Collaborative Tic-Tac-Toe Game

## Overview

This project is a real-time collaborative Tic-Tac-Toe game developed with Django, React (TypeScript), and Node.js (Socket.IO). It offers a dynamic gaming experience by enabling multiple players to play the classic game in real-time. Additionally, the application features an integrated chat system, allowing players to communicate seamlessly during gameplay.

## Key Features

- **Real-Time Gameplay:** Play Tic-Tac-Toe in real-time with multiple players, experiencing dynamic and engaging matches.

- **Chat Functionality:** Communicate with other players using the real-time chat feature, enhancing the collaborative gaming experience.

- **Django Backend:** The backend, powered by Django, handles game logic and server-side operations, ensuring a smooth gaming experience.

- **React with TypeScript Frontend:** The frontend, built with React and TypeScript, provides a responsive and user-friendly interface.

- **Node.js with Socket.IO:** Real-time communication is facilitated by the Socket.IO server built on Node.js, enabling instant updates between the server and clients.

## Usage

1. Players can join ongoing games and participate in real-time tic-tac-toe matches.
2. Utilize the integrated chat feature to communicate and strategize with fellow players.
3. Enjoy the classic game of tic-tac-toe in a collaborative and real-time multiplayer environment.
